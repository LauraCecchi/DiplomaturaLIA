Make: flujo didáctico Excel/Sheets → Gemini → Documento

Construye un escenario simple que toma filas de una planilla, las procesa con Gemini y genera un documento en Google Docs o Word (.docx).

¿Qué vamos a hacer? 1) Crear cuenta en Make 2) Preparar insumos 3) Construir el escenario

¿Qué vamos a hacer?

  • Cuenta en Make.
  • Una planilla: Excel o Google Sheets, con encabezados en la primera fila.
  • Clave de API de Google Gemini (Google AI Studio).
  • Una plantilla de documento con {{placeholders}}.
    • Salida A: Google Docs
    • Salida B: Microsoft Word .docx (guardado en OneDrive)
Tip: Define de antemano qué columnas usarás (ej.: Título, Descripción, Palabras clave).

1) Crear tu cuenta en Make

1
Ve a make.com y elige Sign up. Regístrate con Google o email.
2
Luego, Log in con el mismo método.

¿Ya tienes cuenta?

→
Si ya creaste tu cuenta previamente, simplemente ve a Log in e ingresa con tu usuario. Luego dirígete a Scenarios y pulsa Create a new scenario.
Nota: La versión gratuita tiene límites de operaciones. Para cargas mayores, considera un plan pago.

2) Preparar insumos

Planilla (según selección)

  1. En Microsoft Excel, crea un libro con una hoja y define una Tabla (Ctrl+T) o un rango claro con encabezados.
  2. Inserta algunas filas de ejemplo (p. ej., Título, Descripción, Keywords).
  3. Ten a mano Workbook y Worksheet (los necesitarás en Make).
  1. En Google Sheets, crea una hoja con encabezados en la fila 1 (sin filas en blanco intermedias).
  2. Agrega 2–3 filas de prueba.
  3. Verifica permisos de acceso de tu cuenta de Make a este archivo.
Ejemplo de encabezados 
Nombre | Email| Caso| Fecha

Configurar Gemini (API Key)

  1. Ingresa a Google AI Studio.
    (Necesitarás una cuenta de Google para acceder).
  2. Una vez dentro, busca en el menú la sección “Get API key” o “Obtener clave de API”. Haz clic en ese botón.
  3. Se abrirá una página y verás un botón que dice “Crear una clave de API”.
    4. Haz clic en ese botón y se abrirá una ventana.
  4. Verás un botón “Crear clave de API en un proyecto existente”. Haz clic en ese botón. Google generará una cadena de texto larga, algo parecido a: 
    AIzaSyA1b...tu-clave...Ejemplo123
  5. Cuando se te solicite esta API Key, la copias y la pegas en el lugar que te pidan.
    ⚠️ Importante: No compartas la API Key con otras personas ni la publiques.
  6. Ahora ve a Make → agrega un módulo de Gemini → selecciona “Add connection”.
    Pega tu API Key cuando te la pida.
  7. Haz clic en Save. Si todo salió bien, verás que la conexión a Gemini aparece como “Connected”.

Nota: Si la API Key no funciona, revisa que tu cuenta tenga acceso a Gemini en Google AI Studio y que la clave esté bien copiada (sin espacios extra).
